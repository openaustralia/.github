# .github

This repository holds the organisation-wide GitHub defaults for the [OpenAustralia Foundation](https://oaf.org.au).

## What's in here

- [`profile/README.md`](profile/README.md) - the README shown on our [organisation profile](https://github.com/openaustralia)
- [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) - default issue templates for repositories without their own
- [`.github/PULL_REQUEST_TEMPLATE.md`](.github/PULL_REQUEST_TEMPLATE.md) - default pull request template
- [`.github/FUNDING.yml`](.github/FUNDING.yml) - the "Sponsor" button configuration
- [`CLA/CLA.md`](CLA/CLA.md) - our Contributor Licence Agreement

Templates here apply to any repository in the organisation that doesn't define its own. See the [GitHub documentation on community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for how this works.

## About us

The OpenAustralia Foundation is an Australian charity that builds public digital infrastructure for democracy. We run [Planning Alerts](https://www.planningalerts.org.au), [Right to Know](https://www.righttoknow.org.au), [They Vote for You](https://theyvoteforyou.org.au), [OpenAustralia.org.au](https://www.openaustralia.org.au) and [morph.io](https://morph.io).
